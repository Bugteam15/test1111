
# Shopify Theme - WinRewardsHub

This is a modern, OpenAI-inspired Shopify theme for a sweepstakes and rewards website.

## Theme Features

- Modern, OpenAI-inspired design with dark theme and purple accents
- Glassmorphism UI elements with subtle animations
- Mobile-responsive layout
- Customizable sections and blocks
- Optimized for conversion

## Setup Instructions

1. Download all files in this repository
2. Go to your Shopify admin dashboard
3. Navigate to Online Store > Themes
4. Click "Add theme" > "Upload zip file"
5. Upload the zip file containing all theme files

## Customization

All sections can be customized via the theme editor:
- Hero section - Change heading, description, and button text
- Sweepstakes list - Add/remove sweepstakes entries
- CTA section - Modify call-to-action text and form
- Footer - Update links, contact info, and social media

## Navigation Setup

1. Create navigation menus in Shopify admin (Navigation)
2. Create main-menu for header navigation
3. Create footer menu for footer links

## File Structure

- `/assets` - Contains CSS, JS and other theme assets
- `/sections` - Shopify section files
- `/templates` - Theme templates
- `theme.liquid` - Main theme layout file
